| (4) 주요 기능의 구현-3 | **구매 제안 및 장바구니 기능**<br> 1. 개요 <br>사용자가 등록한 식재료 구매 이력 데이터를 바탕으로, 품목별 구매 주기를 예측해 “다음에 언제, 얼마나 장봐야 하는지”를 추천하는 기능을 구현한다. 이를 위해 **scikit-learn RandomForestRegressor**를 메인 모델로 채택하였으며, 다양한 피처를 바탕으로 예측 정확도를 최적화하였다.<br><br>2. 전체 프로세스<br>- 구매 이력 데이터 로딩<br>- 피처 계산<br>- 시계열 교차검증<br>- 모델 학습 및 예측<br>- 다음 구매일 산출<br>- 추천 결과 제공: 계산된 다음 구매일과 추천 수량을 알림으로 사용자에게 전달<br><br>3. 상세 구현 내용<br>3.1 샘플 데이터 생성<br>- 기간: 12개월(2025-01-01 - 2025-12-31)<br>- 품목별 고정 주기(item_periods: 우유 7일, 계란 10일...)<br>- ±2일 노이즈(jitter) + 포아송(lam=2) 수량<br>- 샘플 데이터프레임 생성 및 정렬<br><br>3.2 데이터 확인<br>- EDA: 날짜별 총 구매량 테이블, 개별 품목 시계열 그래프<br>- 패턴(주기)이 의도대로 생성됐는지 시각화 확인<br><br>3.3 피처 엔지니어링<br>- days_since_last(직전 구매일로부터 경과 일수)<br>- rolling_mean_3(최근 3회 구매 간격 평균)<br>- rolling_median_3(최근 3회 구매 간격 중앙값)<br>- prev_qty(직전 구매 수량)<br>- qty_change_ratio(수량 변화 비율)<br>- period(시뮬레이션/추정 주기)<br>- month(월별 계절성)<br><br>3.4 모델 학습 및 평가<br>- RandomForestRegressor(random_state=42)<br>- TimeSeriesSplit(n_splits=3) + cross_val_score(scoring='neg_mean_absolute_error')<br>- 초기 CV MAE(확장 전): 우유≈1.78일, 계란≈5.67일, 양파≈10.4일<br>- 확장 피처 적용 후 CV MAE: 우유≈0.98일, 계란≈0.84일, 토마토≈2.65일, 양파≈4.20일<br><br>3.5 성능 개선<br>- Baseline 비교: 중위수 예측 대비 평균 30-50% 개선<br>- 샘플 기간 연장: 4→12개월로 데이터 양 확대<br>- 피처 확장: 중앙값·주기·월 정보 추가<br>- 하이퍼파라미터 튜닝: RandomizedSearchCV(n_iter=100) + param_dist 확장, GridSearchCV로 max_features와 max_depth 집중 탐색<br><br>4. 참고<br>- 초기 **Prophet** 검증 결과(우유 3일·계란 5일·양파 11일 MAE)가 허용 기준(±2일)을 벗어나 메인 예측에는 사용하지 않고, 모델 비교·검증용으로만 활용 |
